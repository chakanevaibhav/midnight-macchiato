# Midnight Macchiato Theme

A beautiful, dark VS Code theme inspired by One Monokai and Cappuccino. It is carefully designed for long coding sessions, reducing eye strain while keeping code highly readable and vibrant.

## Features

- **High Readability**: Vivid One Monokai syntax colors that stand out perfectly against the dark backgrounds.
- **Clear Separation**: Distinct background colors for the Activity Bar, Side Bar, Editor, and Terminal to help you easily distinguish different areas of your workspace.
- **Two Variations**:
  - **Midnight Macchiato**: A classic dark theme with warm, rich coffee undertones.
  - **Midnight Macchiato Espresso**: A darker, higher-contrast variation for extreme low-light coding sessions.

## Installation

1. Open **Extensions** sidebar panel in VS Code. `View → Extensions`
2. Search for `Midnight Macchiato`
3. Click **Install** to install it.
4. Click **Reload** to reload your editor
5. Go to File > Preferences > Color Theme > **Midnight Macchiato** or **Midnight Macchiato Espresso**
